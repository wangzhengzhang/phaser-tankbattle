你是一名资深的游戏开发工程师，精通 HTML5 游戏开发。请使用纯 Phaser + JavaScript (ES6+)、HTML 和 CSS 为我创建一个经典的网页版坦克大战游戏。

一、项目核心要求：
1.  **技术栈**：仅使用原生 JavaScript (ES6+) + Phaser 。
2.  **核心玩法**：复刻经典坦克大战的核心玩法，包括玩家坦克移动、射击、摧毁敌方坦克、保护基地。
3.  **视觉风格**：采用像素风，使用简单的色块绘制坦克、子弹、地图障碍物（砖墙、钢墙、水域、森林等）。
4.  **代码质量**：代码模块化，有清晰的注释，易于理解和扩展。
5.  **界面布局**：场景大小：1500x700。顶部有"开始","暂停","重新开始"三个按钮，分别实现相关的点击功能，游戏开始的时候只有"暂停"和"重新开始"按钮可以点击，游戏暂停时只有"开始"按钮可以点击。
6.  **游戏关卡**：一共有10个关卡，难度从第1到10关依次增大，每摧毁一辆地方坦克得分1分，每得50分升1级，进入下一关。
 
8.  **实现游戏地图和碰撞系统**
    绘制关卡地图（例如，用数组表示，0=空地，1=砖墙，2=钢墙，3=水域，4=森林）。
    实现坦克与墙壁、子弹与墙壁、子弹与坦克之间的碰撞检测逻辑。
    不同障碍物有不同效果：
      1）砖墙： 被子弹击中后被摧毁，摧毁之后消失变成空地；
      2）钢墙： 子弹无法摧毁，也不能被坦克穿过；
      3）森林： 坦克可以穿过并隐藏在森林中；子弹不能穿过森林；
      4）水域： 坦克不能穿过水域，子弹可以穿过水域；
      5）空地： 坦克可以自由行使，子弹可以通过；
9.  **实现游戏逻辑**
    计分系统：摧毁敌方坦克得1分。
    生命系统：玩家坦克有多次生命，被击中后重置位置。
    胜利条件：摧毁所有敌方坦克或保护基地一定时间。
    失败条件：玩家生命值为零或基地被摧毁。
    在画布上渲染分数和生命值。
